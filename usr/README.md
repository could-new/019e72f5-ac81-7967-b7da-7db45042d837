# Ticket Bot App

A cross-platform Flutter application for managing and interacting with a support ticket bot. 

## Features
- **Ticket Dashboard**: Responsive list of active and past support tickets.
- **Bot Chat Interface**: Conversational UI to interact with the support bot and resolve issues.
- **Mock Data Models**: Pre-configured with realistic ticket and message models for immediate testing.
- **Cross-Platform**: Designed to work gracefully on Web, Desktop, and Mobile.

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart
- **Architecture**: Stateful Widget based UI with responsive layouts

## Setup and Run
1. Ensure Flutter is installed and set up on your machine.
2. Clone this repository and navigate to the project directory.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to start the app.
